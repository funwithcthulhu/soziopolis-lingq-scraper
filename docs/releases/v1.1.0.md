# Soziopolis Reader 1.1.0

First public Windows build of the tool.

## Highlights

- Browse Soziopolis sections and extract clean article text
- Save articles into a local SQLite library with search, filters, preview, and stats
- Upload saved articles to LingQ with stored credentials and collection selection
- Persist import and upload queue state in SQLite so work survives restarts
- Build a portable Windows folder or a simple Inno Setup installer
- Generate diagnostics bundles with logs, settings, database files, and queue snapshots

## Download

- Windows installer: `SoziopolisReaderSetup-1.1.0.exe`
